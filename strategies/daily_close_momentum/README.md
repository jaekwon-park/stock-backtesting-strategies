# Daily Close Momentum Screener (전일 마감 모멘텀 스크리닝)

전날 1분봉 장 마감 패턴을 분석해 **익일 오전 1% 이상 상승이 기대되는 종목**을 스크리닝하는 전략입니다.

---

## 전략 원리

장 마감 직전에 강한 매수세가 유입된 종목은 다음 날 오전 갭 상승 또는 강한 상승 출발을 보이는 경향이 있습니다. 이 전략은 4가지 조건으로 해당 패턴을 포착합니다.

---

## 진입 조건 (4가지 모두 충족)

| 조건 | 기본값 | 설명 |
|------|--------|------|
| **종가 강도** | ≥ 0.65 | `(종가 - 당일 저가) / (당일 고가 - 당일 저가)` — 1에 가까울수록 고가 근처 마감 |
| **마감 1시간 상승률** | ≥ 0.3% | 마지막 60분(60봉)의 가격 변화율 — 오후 연속 매수세 확인 |
| **마감 30분 거래량 배수** | ≥ 1.5× | 마지막 30봉 평균 거래량 / 당일 전체 평균 거래량 — 세력 개입 여부 |
| **당일 양봉** | True | 종가 > 시가 — 수급 주도 상승 확인 |

---

## 신호 의미

| 신호 | 의미 |
|------|------|
| **BUY** | 전일 4가지 조건 충족 → 익일 오전 1%+ 상승 기대 |
| **SELL** | 이전에 조건을 충족했으나 최근일 조건 미충족 |
| **HOLD** | 조건 미충족 또는 데이터 부족 |

---

## 파라미터

| 파라미터 | 기본값 | 설명 |
|----------|--------|------|
| `min_close_strength` | `0.65` | 종가 강도 하한 (0~1) |
| `min_late_momentum` | `0.003` | 마감 1시간 상승률 하한 (0.003 = 0.3%) |
| `min_vol_surge` | `1.5` | 마감 30분 거래량 배수 하한 |
| `require_bullish_day` | `True` | 당일 양봉 필수 여부 |
| `exit_bars` | `60` | 익일 오전 자동 청산까지 봉 수 (60 = 60분) |

---

## 사용 방법

1. **데이터 수집**: KIS 수집기에서 대상 종목의 **1분봉** 데이터를 최소 3거래일 이상 수집
2. **스크리닝 실행**: 스크리닝 화면에서 아래와 같이 설정
   - Timeframe: `1m`
   - 전략: `[GitHub] daily_close_momentum`
3. **결과 확인**: BUY 신호 종목을 다음 날 오전 장 시작 시 매수 검토

> **참고**: 이 전략은 1분봉 데이터 기준으로 설계됐습니다. 5분봉 이상에서는 신호 정확도가 낮아질 수 있습니다.

---

## 파라미터 조정 예시

조건을 완화해 더 많은 종목을 스크리닝하려면:

```python
context['min_close_strength'] = 0.55   # 종가 강도 완화
context['min_late_momentum'] = 0.001   # 상승률 기준 완화 (0.1%)
context['min_vol_surge'] = 1.2         # 거래량 배수 완화
```

조건을 강화해 정확도를 높이려면:

```python
context['min_close_strength'] = 0.75   # 고가 근처 마감 엄격히
context['min_late_momentum'] = 0.005   # 0.5% 이상 상승
context['min_vol_surge'] = 2.0         # 2배 이상 거래량
```

---

## 주의사항

- 스크리닝 결과는 **참고용**이며 실제 매매 시 추가 검토가 필요합니다.
- 갭 상승 종목은 당일 추격 매수 시 리스크가 높으므로 주의하세요.
- 1분봉 데이터가 없는 종목은 스크리닝에서 제외됩니다.
