유동성(거래대금)과 변동성(ATR) 조건을 동시에 만족하는 종목을 선별하는 스크리닝 전략입니다.

## 개요

일평균 거래대금 300억 원 이상이면서 일봉 ATR/Close 비율이 1.5% 이상인 종목만 통과시킵니다.  
유동성이 충분하고 변동성이 높은 종목을 자동매매·스윙트레이드 대상으로 추려내는 데 활용합니다.

## 스크리닝 조건

| 조건 | 기본값 | 의미 |
|------|--------|------|
| 일평균 거래대금 | ≥ 300억 원 | 최근 20일 `Close × Volume` 평균 — 슬리피지·유동성 확보 |
| ATR(14) / Close | ≥ 1.5% | Wilder 스무딩 ATR ÷ 당일 종가 — 충분한 일중 변동폭 |

## 신호 의미

- **BUY**: 두 조건 모두 충족 → 유동성·변동성 기준 통과 종목
- **SELL**: 조건 하나라도 미달 (보유 포지션 있을 때 청산)

## 파라미터 조정

`initialize` 함수에서 기준값을 변경할 수 있습니다:

```python
context['vol_period'] = 20          # 거래대금 이동평균 기간 (일)
context['atr_period'] = 14          # ATR 기간
context['min_tv'] = 30000000000     # 최소 거래대금 (300억 = 30,000,000,000)
context['min_atr_ratio'] = 0.015    # ATR/Close 최소 비율 (1.5% = 0.015)
```

## 활용 방법

### 유니버스 백테스트 (종목 스크리닝)
- 데이터 수집 완료 후 **유니버스 백테스트** 메뉴에서 이 전략을 선택합니다.
- 수십~수백 종목에 동시 적용하여 조건 충족 종목 목록을 확인합니다.

### 자동매매 세션
- 관심 종목에 대해 **모의투자 세션**을 생성하고 자동매매를 활성화합니다.
- 조건 충족 시 BUY 신호, 미달 시 SELL 신호로 포지션을 관리합니다.

## 데이터 요구사항

- **일봉 데이터**: `market_candles` 테이블에 최소 `vol_period + atr_period` 일(기본 34일) 이상의 일봉 필요
- 데이터 수집 → 시장 데이터 수집(FDR 또는 KIS)으로 일봉 확보 후 사용하세요.
