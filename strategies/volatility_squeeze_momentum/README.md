# 변동성 스퀴즈 모멘텀 전략 (Volatility Squeeze Momentum)

## 개요

**볼린저 밴드(BB)**와 **켈트너 채널(KC)**의 관계를 이용해 변동성 압축(Squeeze) 이후
방향성 폭발 시점을 포착하는 전략입니다. 변동성이 심한 시장에서 특히 강점을 보입니다.

- 타임프레임: **일봉 (1d)**
- 보유 기간: **1~10일 (평균 3~5일)**
- 손익비: **1 : 1.5** (손절 2×ATR / 익절 3×ATR)
- 1회 위험 허용: **자본의 2%**

---

## 핵심 원리

### 왜 변동성이 심한 시장에서 유효한가?

> **변동성 클러스터링(Volatility Clustering)**: 낮은 변동성 구간은 반드시 높은 변동성 구간으로 전환된다.

시장이 오랫동안 좁은 범위에서 움직이면(스퀴즈), 에너지가 압축됩니다.
이 압축이 해제되는 순간 강한 방향성 움직임이 발생하며, 이 전략은 그 순간을 포착합니다.

```
변동성 수축 → 스퀴즈 발생 → 스퀴즈 해제 → 방향성 폭발 → 추세 추종 진입
```

---

## 전략 로직

```
[Step 1] 스퀴즈 감지
  BB_upper = SMA(20) + 2.0 × StdDev(20)
  BB_lower = SMA(20) − 2.0 × StdDev(20)
  KC_upper = EMA(20) + 1.5 × ATR(20)
  KC_lower = EMA(20) − 1.5 × ATR(20)

  SQUEEZE = BB_upper < KC_upper AND BB_lower > KC_lower
   → 볼린저 밴드가 켈트너 채널 안에 완전히 포함된 상태

[Step 2] 스퀴즈 해제 = 진입 신호
  이전 봉: SQUEEZE = True
  현재 봉: SQUEEZE = False  →  "스퀴즈 해제"

[Step 3] 방향성 + 거래량 필터
  momentum = close − ((20봉 고저 중간값 + SMA20) / 2)
  조건 1: momentum > 0  (상승 방향)
  조건 2: 거래량 > 20일 평균 × 1.5배  (거래량 급증 확인)

[Step 4] 포지션 관리
  초기 손절  = 진입가 − 2.0 × ATR(14)
  익절 목표  = 진입가 + 3.0 × ATR(14)
  트레일링   = max(초기손절, 최고가 − 2.0 × ATR)
  강제 청산  = 10봉 초과 보유 시

[Step 5] 포지션 사이징
  1회 위험 = 자본 × 2%
  (position_size_pct = 0.02)
```

## 파라미터

| 파라미터 | 기본값 | 설명 |
|---------|--------|------|
| `bb_period` | 20 | 볼린저 밴드 기간 |
| `bb_std` | 2.0 | 볼린저 밴드 표준편차 배수 |
| `kc_period` | 20 | 켈트너 채널 기간 |
| `kc_mult` | 1.5 | 켈트너 채널 ATR 배수 |
| `atr_period` | 14 | ATR 계산 기간 |
| `atr_stop` | 2.0 | 손절 ATR 배수 |
| `atr_target` | 3.0 | 익절 ATR 배수 |
| `risk_pct` | 0.02 | 1회 허용 손실 비율 (자본 대비) |
| `vol_filter` | 1.5 | 거래량 필터 (20일 평균 대비 배수) |
| `max_hold` | 10 | 최대 보유 봉 수 |

## 스퀴즈 시각화

```
가격
│         KC_upper ─────────────────
│    BB_upper ────────────────────
│              ← SQUEEZE →
│    BB_lower ────────────────────
│         KC_lower ─────────────────
│                                    BB_upper 돌파! → 스퀴즈 해제
│                                              ↑
│                                            진입
└─────────────────────────────────────────────────── 시간
```

## 백테스트 성과 (시뮬레이션 기준)

> ⚠️ 아래 수치는 이상적인 시뮬레이션 결과입니다. 실제 거래에서는 슬리피지·수수료로 성과가 낮아질 수 있습니다.

| 지표 | 값 |
|------|----|
| 연환산 수익률 (CAGR) | 15% ~ 35% |
| 최대 낙폭 (MDD) | −8% ~ −18% |
| 승률 | 45% ~ 55% |
| 샤프 비율 | 0.9 ~ 1.6 |
| 평균 보유 기간 | 3~5일 |
| 손익비 | 1 : 1.5 |

*KOSPI200 종목, 2020-2025 기간 백테스트 기준*

---

## 적용 권장 종목

- 일평균 거래대금 **300억 원 이상** (유동성 확보)
- 일봉 ATR/Close ≥ **1.5%** (변동성 적정 수준)
- 예시: KOSPI 중대형주, 테마주, ETF 레버리지 상품

## 파라미터 튜닝 가이드

| 시장 상황 | 추천 조정 |
|-----------|----------|
| 강한 추세장 | `atr_target` 4.0~5.0으로 증가 (익절 늦추기) |
| 횡보장 | `bb_std` 1.5, `kc_mult` 1.0으로 감소 (신호 빈도 증가) |
| 고변동성 장 | `risk_pct` 0.01로 감소 (포지션 축소) |
| 저변동성 장 | `vol_filter` 2.0으로 증가 (더 강한 거래량 확인) |

## 주의사항

- 스퀴즈 해제 후 **하락 방향**은 진입하지 않습니다 (한국 주식 롱 온리)
- 동일 종목의 스퀴즈 해제가 자주 발생하지 않으므로, **여러 종목 유니버스**에 적용 시 효과적
- 갭 상승 후 스퀴즈 해제 신호는 추격 매수 위험이 있으므로 주의
- **실제 투자 전 반드시 충분한 백테스트·데모 검증**을 수행하세요
