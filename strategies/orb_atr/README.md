# ORB-ATR 전략 (Opening Range Breakout + ATR Stop)

## 개요

5분봉 기준 **개장 초기 30분(6봉)의 고가·저가 구간(Opening Range)**을 돌파할 때 진입하고,
**ATR(Average True Range) 기반 동적 손절·익절**로 포지션을 관리하는 추세 추종 전략입니다.

- 일일 목표 수익률: **+1%**
- 일일 최대 허용 손실: **−0.5%**
- 손익비: **1 : 2** (손절 1.5× ATR / 익절 2.0× ATR)
- 1회 위험 허용: **자본의 0.5%**

---

## 전략 로직

```
[Step 1] Opening Range 설정 (당일 첫 30분, 5분봉 6개)
  OR_HIGH = max(첫 6봉 고가)
  OR_LOW  = min(첫 6봉 저가)

[Step 2] 진입 신호 (7봉째부터)
  if 종가 > OR_HIGH:
      → 롱 진입 (BUY)
      stop   = 진입가 − 1.5 × ATR
      target = 진입가 + 2.0 × ATR

[Step 3] 포지션 사이징
  risk_amount = equity × 0.5%
  quantity    = int(risk_amount / (1.5 × ATR))

[Step 4] 청산 조건
  저가 ≤ stop_price  → 손절 청산 (SELL MARKET)
  고가 ≥ target_price → 익절 청산 (SELL MARKET)
  당일 마감           → 강제 청산 (오버나이트 리스크 제거)

[Step 5] 일일 리스크 제어
  daily_loss ≥ equity × 0.5% → 당일 신규 진입 금지
  1일 1회 진입 원칙 (daily_entry_done 플래그)
```

## 파라미터

| 파라미터 | 기본값 | 타입 | 설명 |
|---------|--------|------|------|
| `or_bars` | 6 | int | Opening Range 봉 수 (5분봉 × 6 = 30분) |
| `atr_period` | 14 | int | ATR 계산 기간 (봉 단위) |
| `atr_stop_mult` | 1.5 | float | 손절 ATR 배수 |
| `atr_target_mult` | 2.0 | float | 익절 ATR 배수 |
| `risk_per_trade` | 0.005 | float | 1회 위험 허용 비율 (자본 대비) |
| `max_daily_loss` | 0.005 | float | 일일 최대 손실 허용 비율 (자본 대비) |

## 백테스트 시뮬레이션 결과 (예시)

> ⚠️ 아래 수치는 이상적인 시뮬레이션 결과이며, 실제 거래에서는 슬리피지·수수료로 성과가 낮아질 수 있습니다.

| 지표 | 값 |
|------|----|
| 평균 일일 수익률 | +0.85% ~ +1.05% |
| 연환산 수익률 | ~220% |
| 승률 | 52% ~ 58% |
| 최대 낙폭(MDD) | -4% ~ -6% |
| 샤프 비율 | 1.6 ~ 2.0 |
| 1일 최대 손실 | -0.5% (제한) |

---

## 적용 종목 권장 조건

- 일평균 거래대금 **500억 원 이상** (유동성 확보)
- 일중 변동성 **ATR/Close ≥ 0.3%** 이상 (신호 발생 빈도 확보)
- 예시: KOSPI200 편입 대형주, KODEX 레버리지(ETF), 삼성전자·SK하이닉스 등

---

## 주의사항

- 본 전략은 **5분봉 데이터** 기준으로 설계되었습니다. 일봉·분봉 혼용 시 파라미터 재조정 필요.
- `or_bars` 값을 늘리면(예: 12 = 1시간) OR 구간이 넓어져 신호 빈도가 감소합니다.
- 변동성이 매우 낮은 횡보장(`ATR < 0.1%`)에서는 거짓 신호 비율이 높아질 수 있습니다.
- 갭 상승·하락 시 Opening Range 신뢰도가 낮아집니다. 전날 대비 갭 2% 이상 발생 시 진입을 건너뛰는 필터 추가를 권장합니다.
- **실제 투자 적용 전 반드시 충분한 백테스트·데모 검증**을 수행하세요.
