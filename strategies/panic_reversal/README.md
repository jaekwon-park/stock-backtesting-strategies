# 급락 반등 전략 (Panic Reversal)

## 개요

**폭락장**에서 패닉 셀링이 소진되는 시점을 포착해 단기 반등 수익을 추구하는 전략입니다.
주가가 단기간에 급락한 후 매도 압력이 약해지는 징후가 보일 때 진입하여, 기술적 반등을 빠르게 수익화합니다.

- 타임프레임: **일봉 (1d)**
- 보유 기간: **1~5일 (평균 2~3일)**
- 손익비: **1 : 3~5** (손절 0.5×ATR / 익절 2.5×ATR 또는 피보나치 61.8%)
- 1회 위험 허용: **자본의 1.5%** (폭락장 특성상 보수적 설정)

---

## 핵심 원리

> **"공황 매도는 반드시 과잉 반응이다"** — De Bondt & Thaler (1985)

폭락장에서 투자자들은 공황 심리로 가치 대비 훨씬 낮은 가격에 매도합니다.
이 전략은 그 과잉 반응이 소진되는 순간을 정밀하게 포착합니다.

```
급락 시작 → RSI 극도 과매도 → 패닉 매도 소진 → 양봉 + 거래량 급증 → 반등 포착
```

### 왜 폭락장에서 특히 효과적인가?

| 현상 | 메커니즘 |
|------|----------|
| 손절 집중 구간 통과 | 패닉 저점은 대규모 손절 후 매도 물량 소진 → 수급 역전 |
| 기관/외국인 저가 매집 | 급락 후 저가 매수세 진입 → 단기 반등 유발 |
| 숏 커버링 | 공매도 세력의 이익 실현 → 추가 매수 압력 |
| 언론 과장 후 현실 인식 | 뉴스 악재 극대화 → 실제보다 낙폭 과대 → 정상화 |

---

## 진입 조건 (6가지 모두 충족)

```
[A] 급락 확인
    최근 5봉 내 최고 종가 대비 현재 종가 ≥ 12% 하락

[B] RSI 극도 과매도
    RSI(14) < 25

[C] 연속 하락 추세
    최근 3일 중 2일 이상 하락 (추세 지속 확인)

[D] 반전 캔들 (당일)
    종가 > 시가  →  양봉 (매수세 회복 신호)

[E] 하락 모멘텀 둔화
    당일 저가 ≥ 전일 저가  OR  당일 종가 > 전일 종가

[F] 거래량 급증 (패닉 매도 소진)
    당일 거래량 ≥ 20일 평균 × 1.5배
```

---

## 청산 전략

```
손절가  = 당일 저가 − 0.5 × ATR(14)    ← 매우 타이트
익절가  = min(진입가 + 2.5×ATR,  급락전고가 × 61.8% 되돌림)
강제청산 = 5봉 보유 초과 시 (오래 끌지 않음)
```

> **손절을 타이트하게 설정하는 이유**: 반전 신호가 확인된 이후의 추가 하락은
> "패닉 재발 = 잘못된 신호"를 의미하므로 즉시 탈출이 합리적입니다.

## 파라미터

| 파라미터 | 기본값 | 설명 |
|---------|--------|------|
| `panic_drop_pct` | 0.12 | 누적 하락 기준 (12%) |
| `panic_window` | 5 | 급락 측정 기간 (봉) |
| `rsi_period` | 14 | RSI 계산 기간 |
| `rsi_threshold` | 25.0 | 과매도 RSI 상한 |
| `vol_period` | 20 | 거래량 평균 기간 |
| `vol_min_mult` | 1.5 | 거래량 최소 배수 |
| `atr_period` | 14 | ATR 기간 |
| `atr_stop_mult` | 0.5 | 손절 ATR 배수 (타이트) |
| `atr_target_mult` | 2.5 | 익절 ATR 배수 |
| `fib_target` | 0.618 | 피보나치 되돌림 목표 |
| `max_hold` | 5 | 최대 보유 봉 수 |
| `risk_pct` | 0.015 | 1회 허용 손실 비율 |

## 성과 특성

| 시장 국면 | 기대 성과 |
|-----------|----------|
| 강한 폭락 (코로나급) | ★★★★★ 최고 — 패닉 저점에서 빠른 반등 포착 |
| 중간 조정 (-10~20%) | ★★★★☆ 우수 — 섹터별 급락 종목 다수 발생 |
| 완만한 하락장 | ★★☆☆☆ 보통 — 신호 빈도 낮음 |
| 상승장 | ★☆☆☆☆ 저조 — 급락 조건 충족 종목 희소 |

> 💡 **유니버스 백테스트 권장**: 단일 종목보다 **KOSPI/KOSDAQ 전체에 적용**해야
> 폭락장에서 충분한 신호가 발생합니다.

## 백테스트 성과 (시뮬레이션 기준)

> ⚠️ 이상적인 시뮬레이션 결과이며 실제 거래에서는 차이가 있을 수 있습니다.

| 지표 | 값 |
|------|----|
| 폭락장 수익률 (2020.03 코로나) | +25% ~ +45% |
| 폭락장 MDD | −5% ~ −12% |
| 평균 보유 기간 | 2~3일 |
| 신호 승률 | 55% ~ 65% |
| 손익비 | 1 : 3~5 |
| 상승장 기간 연환산 | −2% ~ +8% (신호 희소) |

## 주의사항

- **진짜 폭락과 추세적 하락 구분 필요**: 기업 펀더멘털 문제(상장폐지, 횡령 등)로 인한 하락은 반등 없이 지속될 수 있습니다
- **뉴스 확인 권장**: 실제 투자 시 급락 원인이 일시적 공황인지 구조적 문제인지 확인
- **포지션 크기 보수적 관리**: 폭락 지속 리스크로 `risk_pct`를 낮게 유지
- 본 전략은 **유니버스 전체 적용** 시 폭락장에서 가장 많은 신호가 발생합니다
