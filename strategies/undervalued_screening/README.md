재무 지표(PER·PBR·ROE·EPS) 기반으로 저평가된 종목을 선별하는 스크리닝 전략입니다.

## 개요

KIS 재무데이터 수집 또는 DART 재무제표 수집 후 **종목 스크리닝** 메뉴에서 사용합니다.  
각 종목의 최신 재무 데이터를 기반으로 저평가 여부를 판단합니다.

## 저평가 판단 기준

| 지표 | 기본값 | 의미 |
|------|--------|------|
| PER  | ≤ 15.0 | 이익 대비 주가 — 낮을수록 저평가 |
| PBR  | ≤ 1.2  | 장부가치 대비 주가 — 1 미만이면 청산가치 이하 |
| ROE  | ≥ 8%   | 자기자본이익률 — 수익성 기준 |
| EPS  | ≥ 1    | 적자 기업 제외 |

## 신호 의미

- **BUY**: 모든 기준 충족 → 저평가 종목
- **SELL**: 기준 미충족 (보유 포지션 있을 때 청산)
- **HOLD**: 조건 미충족 또는 재무 데이터 없음

## 파라미터 조정

`initialize` 함수에서 기준값을 변경할 수 있습니다:

```python
context['max_per'] = 15.0   # PER 상한
context['max_pbr'] = 1.2    # PBR 상한
context['min_roe'] = 8.0    # ROE 하한 (%)
context['min_eps'] = 1.0    # EPS 최솟값
```

## 데이터 요구사항

1. **KIS 재무데이터 수집** (추천): 데이터 수집 → KIS (재무데이터) → 전체 종목 수집
2. **DART 재무제표 수집**: 데이터 수집 → DART (재무제표) → 연간 데이터 수집

재무 데이터가 없는 종목은 자동으로 HOLD 처리됩니다.
